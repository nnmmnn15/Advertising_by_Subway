## 💡 핵심 인사이트

- **20대**는 오후 ~ 심야(17 ~ 24시) 집중, 주말·방학 기간에 유동인구 급증
- **30/40대**는 출퇴근 시간대(7 ~ 9시, 17 ~ 19시) 피크, 평일 패턴 뚜렷
- **50대 이상(우대권)**은 오전(10 ~ 14시) 집중, 요일 편차 작음
- **방학 기간(vac=1)** 20대 유동인구 평균 대비 약 10 ~ 17% 증가
- COVID-19 기간 데이터를 별도 처리해 이상치 영향 최소화

→ **시간대·요일·연령대 조합에 따라 광고 타겟을 자동 추천하는 모델 구현**

## 📁 담당 파트 코드 경로

```
Advertising_by_Subway/
│
└── DataAnalysis/
    └── 홍대입구/              ← 담당 파트
        ├── Analysis/
        │   ├── data 확인.ipynb     # 원본 데이터 탐색 및 구조 파악
        │   ├── 1220.ipynb          # 승객유형 4종 통합 전처리
        │   ├── 1222.ipynb          # pivot + 요일/시간대/방학 분류
        │   └── 1224.ipynb          # 20세 미만 심층 분석 및 시각화

            ├── random_forest.joblib

```

## 🔍 분석 흐름

```
원본 CSV (승하차 + 승객유형)
    ↓ data 확인.ipynb
    탐색 · 요일 컬럼 추가 · pivot 구조 확인
    ↓ 1220.ipynb
    승객유형 12종 → 일반 / 청소년 / 우대권 / 외국인 4종 통합
    ↓ 1222.ipynb
    pivot_table · vac(방학) 컬럼 · 요일/시간대별 분류
    ↓ 1224.ipynb
    20세 미만 집중 분석 · melt → long format · 시각화
    ↓ Modeling
    MultiOutputRegressor + RandomForest → 광고 타겟 추천
```
